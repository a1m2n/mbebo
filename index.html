Human Robot Interaction Project

Languages: C/C++

In this project developed with an arduino uno kit in the Arduino IDE, I attempted to give emotions to a robot I made. I experimented with how to convey emotions through movement, sound and light. I asked a third-person to guess the emotions after sometime. I discoovered that some basic emotions such as angry, sad, friendly and happy where easy to convey with movement, sound and light.

UnderWater Game

Languages: Python, Kivy

This project is a simple game developed in the kivy module of python. I made the game after learning how to code with kivy on youTube. In the game you have to control a sprite using WASD as you move along a specified track - similar to an endless runner game but much simplified. The game ends when you miss a track. Think of the piano tiles game but instead of music you controlling a sprite on a continuous flow of tiles and earning points.

Data Visualization: Mapping Covid19 Data

Languages: JavaScript, p5js

In this project, I obtained open source data about the spread of Covid19 since 2020 John Hopkins University Center for System Science and Engineering. Using mapbox API, I decided to visulaize the spread of covid19 in p5js. I used Web Mercator to convert each country's longitude and latitude positions into x and y coordinates. I tried this project after learning about creative coding in p5js from Daniel Shiffman on YouTube.
